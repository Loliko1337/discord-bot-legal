Datenschutzerklärung

Diese Erklärung informiert dich darüber, wie dieser Discord-Bot („Bot“) personenbezogene Daten verarbeitet.

1. Welche Daten werden erfasst?

Der Bot kann folgende Informationen verarbeiten: Benutzer-ID, Server-ID, Nachrichteninhalte (z. B. für Befehle), Rollen und Reaktionen. Es werden keine sensiblen personenbezogenen Daten gespeichert.

2. Zweck der Datenverarbeitung

Daten werden ausschließlich zur Funktionalität des Bots verwendet (z. B. für Moderation, Automatisierungen, Servereinstellungen).

3. Speicherung

Daten werden nur so lange gespeichert, wie sie für die Bot-Funktionen notwendig sind, und können auf Anfrage gelöscht werden.

4. Weitergabe

Es erfolgt keine Weitergabe deiner Daten an Dritte, außer wenn gesetzlich vorgeschrieben.

5. Kontakt

Bei Fragen oder zur Löschung deiner Daten wende dich bitte an den Bot-Entwickler: @LolikoHD (z. B. über Discord).
